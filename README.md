# One Cloud Coffee & Eatery Chat App

A real-time chat application for One Cloud Coffee & Eatery.

## Pages
- **User Chat App** (`/index.html`): For users to login and chat
- **Display Page** (`/display.html`): For projector display
- **Admin Page** (`/admin.html`): For administration

## Setup Firebase
1. Create a Firebase project
2. Enable Realtime Database
3. Update `public/firebase-config.js` with your Firebase credentials

## Installation
```bash
npm install
```

## Run Locally
```bash
npm run dev
```

## Deploy to Vercel
1. Push to GitHub
2. Import repository in Vercel
3. Deploy!
